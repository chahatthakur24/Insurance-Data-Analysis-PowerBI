# Insurance Data Analysis Dashboard

## Overview

An interactive Power BI dashboard developed to analyze insurance policy performance, customer demographics, premium trends, and claim distribution.

## Tools Used

- Power BI
- DAX
- MS SQL Server
- Power Query
- Data Modeling

## Business Problem

Insurance companies generate large volumes of policy and claims data. This dashboard helps monitor key business metrics and identify patterns in premiums, claims, and customer behavior.

## Dashboard Features

### KPIs

- Total Premium Amount
- Total Coverage Amount
- Total Claim Amount

### Visualizations

- Premium Amount by Policy Type
- Claims by Status
- Customer Activity Analysis
- Claim Amount by Age Group
- Policy Performance Metrics
